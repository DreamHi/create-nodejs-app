# template-nodejs-app
基于Nodejs的后端服务器模版，只提供RestAPI服务。适用于前后端分离的项目。
前端模版请参考[template-react-app]()

## 环境要求

node > 6
mongodb > 3.6

## 

## 运行
```sh
    // 1.启动mongodb
    cd ./mongodb-3.6.3
    bin/mongod --dbpath=./data/
    // 2.安装依赖的库
    npm install
    // 3.启动服务
    node app.js 
    
    // 4.检证

```
